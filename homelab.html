Homelab

My homelab is used to host some service and learn network and system administration about network and system administration.
I operate Proxmox as hypervisor, Debian and RHEL for the virtual machines. I use HAproxy, Nginx and apache in reverse proxy and to host web service. I monitor my stuff with Prometheus and Grafana. I like automatization, i use Packer, Terraform and Ansible for that. A partial of my configuration is available on : gitlab
The network is managed by Mikrotik and Ubiquiti hardware. In addition to serving as a continuous learning experience, i host some service for I Learned association. I am also a supporter of "self-hosted" mind, as a result i host most of the services i use.
